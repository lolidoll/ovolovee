# 朋友圈功能 - 快速使用指南

## 🎯 核心功能

### 发布朋友圈
```
1. 点击 "分享新鲜事..."
2. 输入文字
3. 点击 📷 添加图片（可选）
4. 选择分组可见范围
5. 点击 "发布"
```

### 修改用户信息
```
点击用户名 → 修改用户名
点击访客数字 → 修改访客总量
```

### 互动
- **点赞**：点击朋友圈下的"点赞"
- **评论**：输入框输入 + 回车，或点击"评论"按钮
- **转发**：点击"转发"选择好友

## 🎨 功能菜单

| 按钮 | 功能 |
|------|------|
| 说说 | 打开发布对话框 |
| 日志 | 日志功能（扩展） |
| 相册 | 相册功能（扩展） |
| 留言 | 留言功能（扩展） |
| 更多 | 打开更多选项菜单 |

## 📋 更多选项

### 角色发布朋友圈
- 选择好友角色
- 选择自动生成或手动输入内容
- 发布后以该角色名义显示朋友圈

### 自动生成朋友圈
- 设置生成间隔时间（分钟）
- 设置每次生成数量
- 启用/禁用自动生成

### 自动回复设置
- 启用后，AI会自动回复别人的评论

## ⚙️ 其他功能

- **通知**：点击顶部"通知"按钮查看
- **设置**：点击顶部"设置"按钮更改背景

## 💾 数据保存

所有朋友圈、评论、通知等数据都自动保存到本地localStorage中，刷新页面后数据依然存在。

## 🔄 同步机制

用户头像、名字、访客总量每5秒自动与侧边栏用户信息同步一次。修改任意一侧的信息，另一侧会自动更新。

## 📝 数据结构

```javascript
// 单条朋友圈
{

  visibility: "group_all", // 可见范围
  isUserPost: true, // 是否用户发布
  createdAt: "2026-01-21T...",
  likes: 0,
  liked: false
}
```
