## 💻 技术实现

### 集成方式

```
index.html 加载
  ↓
引入 auth-modal.css 和 auth-modal.js
  ↓
在 body 中添加模态框 HTML
  ↓
AuthModalManager 自动初始化
  ↓
根据登录状态显示/隐藏
```

### 两个系统的协作

```
login.js (OAuth 认证)
  ↓
authManager.isUserLoggedIn()
  ↓
auth-modal.js (UI 管理)
  ↓
显示/隐藏模态框
```

## 🚀 使用方式


### 手动控制

```javascript

authModalManager.hide();        // 隐藏
```
