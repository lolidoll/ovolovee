### 2️⃣ 猫耳装饰 ✨
```css

.auth-header::before  → 左猫耳
.auth-header::after   → 右猫耳

特性：
- 三角形边框技巧实现
- 粉色三角形（#ff69b4）
- drop-shadow 柔和阴影
- 响应式调整大小
```

### 3️⃣ 蕾丝边装饰 👑
```css
/* 功能卡片边框 */
.auth-intro-item::before {
    repeating-linear-gradient(
        90deg,
        transparent 0px,
        transparent 10px,
        rgba(255, 182, 193, 0.2) 10px,
        rgba(255, 182, 193, 0.2) 12px
    )
}

效果：
- 垂直条纹蕾丝纹理
- 半透明粉色
- 叠加在卡片上
- 增加少女感
```

### 4️⃣ 渐变和阴影
```
文字渐变：
- .auth-title: linear-gradient(#ff69b4 → #ffb6c1)
- 使用 background-clip: text 实现

边框和阴影：
- 内层阴影：inset 粉色
- 外层阴影：rgba(255, 105, 180, 0.15)
- 增加柔软感觉
```
