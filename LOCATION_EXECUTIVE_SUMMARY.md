## 🎯 使用示例

### 用户发送地理位置
```
1. 点击 📍 按钮
2. 输入：天安门广场
3. 地址：北京市东城区
4. Ctrl+Enter 或点击发送
5. 消息显示在对话中
```

### AI发送地理位置
```
AI在回复中：
我建议去【地理位置】故宫|北京市东城区景山前街4号【/地理位置】

系统自动识别并显示为：
📍
故宫
北京市东城区景山前街4号
```
